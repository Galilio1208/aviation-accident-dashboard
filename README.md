\# Aviation Accident Dashboard



\## Project Overview

The Aviation Accident Dashboard is an interactive Excel-based data analysis project designed to explore aviation accident records and identify meaningful patterns across time, location, purpose of flight, phase of flight, and aircraft damage.



This dashboard transforms raw aviation data into easy-to-understand visual insights using pivot tables, charts, slicers, and KPI cards. The goal of the project is to support analytical decision-making and present accident trends in a clear, professional format.



\## Objectives

\- Analyze aviation accident trends over time

\- Identify locations with higher accident occurrence

\- Study accidents by purpose of flight

\- Examine accident distribution across phases of flight

\- Understand aircraft damage patterns

\- Build an interactive Excel dashboard for analysis



\## Tools Used

\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Slicers

\- Timeline Filters

\- KPI Cards

\- Data Cleaning and Formatting



\## Dataset

This project uses aviation accident records containing fields such as:

\- Event Date

\- Location

\- Country

\- Injury Severity

\- Aircraft Damage

\- Aircraft Category

\- Purpose of Flight

\- Weather Condition

\- Broad Phase of Flight

\- Total Fatal Injuries

\- Total Serious Injuries

\- Total Minor Injuries

\- Total Uninjured



\## Dashboard Features

\- KPI summary section for high-level metrics

\- Year-wise accident trend analysis

\- Location-wise accident analysis

\- Purpose of flight breakdown

\- Phase of flight analysis

\- Aircraft damage analysis

\- Interactive slicers and filters for dynamic exploration
